# Nike Frontend Layout (React.js)

This is a dummy front-end project where I recreated the layout of the Nike website using React.js.  
It includes responsive design and component-based structure using React Router.

## Features
- Home, About, Products, and Contact pages
- Responsive design using CSS
- React Router navigation

> Note: This project is front-end only and does not include backend functionality.

## How to run
1. Clone the repo
2. Run `npm install`
3. Run `npm start`
